# 第6章 跨域实证：从感知到行动

第1-5章完成了YLYW从哲学根基到架构设计到学习机制到安全约束的完整理论体系。但理论的最终验证不是逻辑推导——是实验。

本章报告YLYW在三个主要验证域——物理域（YCB物体抓取）、运动控制域（人形机器人步态）、具身导航域（ALFWorld）——以及多个补充域（视觉分类、书法学习、嵌入式部署）上的系统实验。每项实验遵循统一的设计原则：零样本优先、消融验证、对比评估、可解释追溯。

---

## 6.1 实验设计的整体思路

### 6.1.1 为什么跨域——验证架构的通用性

YLYW架构的核心主张之一是：八卦隶属度→六爻编码→卦象匹配的三层推理架构是域无关的。这不仅仅是一个理论声明——它必须通过跨域验证来检验。

如果YLYW仅在物理抓取域有效，可以解释为"64卦恰好适合描述物体的物理属性"，但这种巧合解释的效力是有限的。只有当相同架构（仅替换各层的语义内容）在完全不同域——运动和交互、导航和探索、感知和分类——中同样有效时，才能证明架构本身具有通用性。

跨域迁移的工程含义是：每次迁移只改变L1原型向量、L2编码公式和L3规则库的语义锚点，**架构骨架、推理流程和参数更新机制保持不变**。这意味着在物理域开发的所有工具——余弦相似度匹配、爻位关系运算、双八卦仲裁、知几知耻学习——在运动域和导航域中无需修改即可复用。

### 6.1.2 四原则

每项实验遵循四原则。

**零样本优先。** 不进行任何参数调整或训练数据注入，以完全初始的六十四卦规则进行推理。这是系统的"底线测试"——它回答"先验知识在没有数据的情况下能提供多少决策能力"。

**消融验证。** 每个设计选择（易理 vs 随机映射、三层 vs 仅L3、连续 vs 硬阈值）通过A/B测试量化其独立贡献。第2-3章报告的三维消融实验覆盖物理域；安全八卦消融覆盖安全域。

**对比评估。** 在可能的情况下，与域内的基准方法进行定量比较。在物理域，YLYW的92.0%零样本成功率与Dex-Net（93%）、GraspNet（65-85%）和GG-CNN（83%）进行间接对比。在ALFWorld导航域，YLYW的_73.9%/90.0%（全集/一致子集）与BUTLER（37%）、Reflexion（77%）和EmbodiSkill（93.28%）进行直接对比。

**可解释评估。** 不仅报告宏观指标（成功率、平均分），还输出可追溯的推理链案例。推搡恢复实验的完整卦象切换链（5个卦象、4秒恢复）、YCB花瓶100%成功率的物理解释（角度增益使几何锁止补偿低摩擦）——每个案例的推理链都是可读的。

### 6.1.3 评估指标的定义

不同域使用不同的评估指标，但共享一个设计原则——多维度独立评估。

| 域 | 指标 | 说明 |
|:--|------|------|
| 物理抓取 | 成功率 | 力学模型总分≥0.50为成功 |
| | 安全率 | 安全维度得分≥0.50为安全 |
| | 子维度分 | 力闭合/提升/安全的独立得分 |
| 运动控制 | 姿态稳定性 | IMU倾角的归一化值 |
| | 恢复时间 | 从扰动到正常步态的用时 |
| | 卦象切换链 | 完整的情境过渡序列 |
| ALFWorld | 任务成功率 | PDDL标注判定成功 |
| | 平均步数 | 每局探索步数 |
| | 标注一致率 | 环境标注与任务描述的一致性 |
