# Version History

## v1.0.0 (2026-06-25) — 初始公开发布

### 核心功能

**三层推理架构 (L1→L2→L3→L3+)**
- **L1 八卦隶属度层**: 将13维物理特征通过高斯核映射为8维连续隶属度向量，实现从连续物理量到符号空间的零信息损失映射
- **L2 六爻编码层**: 6条加权聚合公式将8维隶属度重组为6维语义爻值（稳定性/可达性/力需求/脆弱性/优先级/环境约束），每爻可判定阴阳
- **L3 六十四卦匹配层**: 余弦相似度在64卦爻模板空间中搜索最佳匹配卦象，激活对应预定义策略（39种策略类型，覆盖全部基本操作原语）
- **L3+ 爻位关系运算**: 将《周易》"乘承比应当位得中"形式化为5种可计算评分算子，输出综合爻位质量和力修正系数

**先验手册 (PriorManual)**
- 统一推理接口：`manual.process(features) → (perception, strategy)`
- 完整可解释推理链：`manual.explain_reasoning()`
- 批量推理引擎：`ylyw_engine.py` CLI
- 总参数量：443个，单次推理~1.7ms (CPU)

**知识体系**
- 八卦物理原型映射表（乾☰=刚健、坤☷=柔顺、震☳=动态、艮☶=静止、离☲=可见、坎☵=危险、兑☱=柔和、巽☴=入微）
- 64卦爻模板库（每个卦的6维理想爻向量，含余弦相似度匹配）
- 64卦→策略映射表（39种抓取策略 + 力/速度/角度/注意事项）
- 40+种物体预设特征（YCB标准物体 + 扩展物体）

### 已知局限（详见专著第10章）
1. 感知前端缺失：当前13维特征来自仿真/人工标定，真实RGB→特征Adapter未实现
2. 先验边界：64卦在非物理域（视觉、语言）覆盖率有限
3. 学习天花板：443参数将学习上限锁定在~95%物理合理率
4. 语言爻编码：语义爻编码公式缺乏系统化定义
5. 连续控制未验证：YLYW在力矩级连续控制上的适用性待探索
