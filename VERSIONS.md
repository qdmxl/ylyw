# YLYW 版本日志

## 版本命名规范

```
实验版本: exp/<功能描述>
论文版本: paper/v<主>.<次>

```

每次 `git tag` 对应一个可回溯的检查点。

## 快速回溯命令

```bash
# 查看所有版本
git tag -l

# 回到某个版本
git checkout <tag>

# 查看两个版本间代码差异
git diff release/v1.0..release/v2.0 -- prior_manual/

# 查看两个版本间实验数据差异
git diff release/v1.0..release/v2.0 -- baseline_50objects_report.json
```
