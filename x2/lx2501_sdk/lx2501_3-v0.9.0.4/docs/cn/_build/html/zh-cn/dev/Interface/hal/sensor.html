5.4.1 传感器接口

传感器接口提供了机器人各种传感器的数据获取和控制能力，包括相机、IMU、激光雷达、触摸传感器等。

核心功能

视觉传感器

  • RGB相机：彩色图像获取

  • 深度相机：深度信息获取

  • 相机内参：标定参数获取

姿态传感器

  • IMU数据：加速度、角速度、姿态角

  • 陀螺仪：角速度测量

  • 加速度计：加速度测量

环境感知传感器

  • 激光雷达：点云数据获取

  • 触摸传感器：触觉反馈

标准传感器消息

传感器接口大多使用ROS标准传感器定义sensor_msgs中的消息类型:
